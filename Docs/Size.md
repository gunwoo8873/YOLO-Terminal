# Size Docs

## Application Window Size

```rust
struct WindowSize {
  width: f32,
  height: f32,
}

fn main() -> Result {
  let size = WindowSize {
    width: 320.0,
    height: 240.0,
  };

  let window_width = size.width;
  let window_height = size.height;
}
```
